## v1.4.5: GitHub連携カードのUI刷新とAI深掘り詳細検索

AIアシスタント画面でGitHubリポジトリを検索した際に表示されるカードのUI表現力を強化し、さらにAIへの深掘り質問をしやすくしたアップデートです。

### 🌐 GitHub連携カードのUI刷新とAI深掘り詳細検索
* **オーナーアバターの表示**: Coil（`AsyncImage`）を用いてリポジトリオーナーの実際のプロフィールアバター画像を非同期取得し、プレースホルダーの代わりに表示するように変更しました。
* **AI深掘り詳細検索アクション**: 各リポジトリカードの右下に「詳細検索（虫眼鏡）」ボタンを追加。タップするだけで、「リポジトリ『owner/repo』のさらに詳細な情報を教えてください」という対話クエリが自動送信され、AIに該当プロジェクトの機能や README 解説の要約をスムーズに深掘り質問できるようになりました。
* **ブラウザ遷移の安全設計**: ブラウザを起動してリポジトリURLへ遷移する（`ACTION_VIEW`）際の例外処理を `try-catch` で多重ガードし、ブラウザアプリが未設定な環境でもクラッシュせず安全に警告（トースト）を表示する機構を導入しました。
