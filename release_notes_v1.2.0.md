## v1.2.0: 高速一括AI解析機能 (Bulk Analysis) & 設定の改善

AI機能の向上と操作性の最適化を図ったアップデートです。

### ⚡ 高速一括AI解析機能 (Bulk Analysis)
* **選べる解析モード**: 未解析のアプリをまとめて解析する際、1回のリクエストで複数（最大10件）のアプリを高速に解析・分類できる「高速一括解析 (バルク)」機能を追加しました。
* 解析開始時に、従来の丁寧な「個別解析 (順次)」か、高速な「一括解析 (バルク)」かを選択するダイアログを導入しました。

### ⚙️ 設定画面の改善
* **主要モデルのクイック選択**: プライマリおよびバックアップモデルの入力エリアの下に、`gemini-flash-latest` と `gemini-flash-lite-latest` をワンタップで選べる簡易切り替えボタンを追加しました。

### 🤖 Gemma モデル対応 & 堅牢化
* ツール呼び出し（Function Calling）や構造化出力に対応していない Gemma モデルでの動作を最適化し、安定したAI動作を実現しました。
* Gemini API からの JSON レスポンスが Markdown ブロックで囲まれている場合等の JSON パース処理をさらに堅牢にしました。

### ⚠️ Rate Limit エラーハンドリングの強化
* Gemini API 無料枠の回数制限（HTTP 429）に達した際のエラーメッセージを刷新し、具体的な原因や解決策（時間をおく、自分のAPIキーを使用するなど）をわかりやすく案内し、設定画面への誘導を追加しました。
